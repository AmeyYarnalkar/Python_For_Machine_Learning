## What is a List in Python?

A **list** is just a **container** that can store multiple values in one place.

Instead of making many variables like:

```
a = 10
b = 20
c = 30
```

You can store everything together:

```python
numbers = [10, 20, 30]
```

Think of a list like a row of boxes:

```
[ 10 | 20 | 30 ]
```

Each box has a position.

---

## Main Characteristics 

### Lists keep order

```python
[10, 20, 30]
```

10 comes first, then 20, then 30.

---

### Lists can change (Mutable)

You can modify values later.

```python
numbers[0] = 99
```

Now list becomes:

```
[99, 20, 30]
```

---

### Lists can store anything

```python
data = [10, "hello", True, 4.5]
```

Numbers, text, yes/no values ‚Äî all allowed.

---

### Lists can grow or shrink

You don‚Äôt decide size in advance.

---

## When Should You Use Lists?

Use lists when:

‚úî You want to store many items together\
‚úî Order matters\
‚úî You want to change values later

Example:

* marks of students
* names
* steps of a process

---

## How Lists Are Stored in Memory 

Python does **not** store values directly inside the list.

Instead, it stores **references** (like arrows pointing to values).

Imagine:

```
List ‚Üí [ ‚Üí10 , ‚Üí20 , ‚Üí30 ]
```

Why this matters:

* Lists can store mixed data types.
* Changing one item doesn‚Äôt affect others.

You don‚Äôt need deep memory knowledge yet ‚Äî just know lists are flexible containers.

## How to Create a List

### Method 1 ‚Äî Using square brackets `[ ]`

```python
numbers = [10, 20, 30]
```

Empty list:

```python
numbers = []
```

---

### Method 2 ‚Äî Using `list()` function

```python
numbers = list()
```

Convert text into list:

```python
letters = list("abc")
# ['a', 'b', 'c']
```

---

## How to Access Elements

Every item has a **position number** called index.

Index starts from **0**.

```
[10, 20, 30]
 0   1   2
```

Access like this:

```python
numbers[0]   # gives 10
numbers[1]   # gives 20
```

---

### Access Last Element

```python
numbers[-1]
```

Negative index means counting from the end.

---

## Add and Update Elements

### Add element at end

```python
numbers.append(40)
```

Result:

```
[10, 20, 30, 40]
```

---

### Add at specific position

```python
numbers.insert(1, 15)
```

Result:

```
[10, 15, 20, 30]
```

---

### Update value

```python
numbers[0] = 100
```

---

## Check if Element Exists

Use keyword `in`.

```python
if 20 in numbers:
    print("Found")
```

Python checks each item until it finds it.

---

## Delete Elements

### Remove by value

```python
numbers.remove(20)
```

Removes first 20 it finds.

---

### Remove by position

```python
numbers.pop()
```

Removes last element.

```python
numbers.pop(1)
```

Removes element at index 1.

---

### Delete using `del`

```python
del numbers[0]
```

---

## How to Go Through (Iterate) a List

### BEST Way 

```python
for item in numbers:
    print(item)
```

This reads naturally:

üëâ ‚ÄúFor each item in numbers, print it.‚Äù

---

### When you need index + value

```python
for i, value in enumerate(numbers):
    print(i, value)
```

---

##  Not Recommended 

```python
for i in range(len(numbers)):
    print(numbers[i])
```

It works ‚Äî but it‚Äôs more complex and easier to make mistakes.

---

## Copying a List

Wrong way:

```python
a = [1,2,3]
b = a
```

Both point to SAME list.

Correct way:

```python
b = a.copy()
```
