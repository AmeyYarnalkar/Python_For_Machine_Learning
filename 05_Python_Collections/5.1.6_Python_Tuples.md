

## What is a Tuple?


A **tuple** is a collection of items just like a list, but:

ðŸ‘‰ **It cannot be changed after creation** (immutable).

Example:


Once created:


Simple idea:


## How is a Tuple Represented?

Tuples use **round brackets** `()`.

```python
numbers = (1, 2, 3)
```

Visual:

```
( 1 , 2 , 3 )
```

Important:

A tuple is still ordered like a list.

```
(10, 20, 30)
 0   1   2
```

##  How to Create a Tuple

### Basic creation

```python
t = (1, 2, 3)
```

### Empty tuple

```python
t = ()
```

### Single element tuple (VERY IMPORTANT)

```python
t = (5,)     # comma is required
```

Without comma:

```python
t = (5)      # this is just number 5, not tuple
```

---

### Create tuple without brackets (packing)

```python
t = 1, 2, 3
```

Python automatically makes it a tuple.

---

## How to Access Elements

Tuples use indexing like lists.

```python
nums = (10, 20, 30)

print(nums[0])   # 10
print(nums[1])   # 20
```

Negative index:

```python
print(nums[-1])  # last element
```

---

## How to Iterate Over Tuples

### BEST way (simple loop)

```python
nums = (10, 20, 30)

for item in nums:
    print(item)
```

---

### With index

```python
for i, value in enumerate(nums):
    print(i, value)
```

## Common Mistakes Beginners Make

###  Trying to modify tuple

```python
nums = (10, 20, 30)
nums[0] = 100   # ERROR
```

Tuples are immutable.

---

### Forgetting comma in single-element tuple

```python
t = (5)     # NOT a tuple
t = (5,)    # correct
```

---

###  Thinking tuples save deep immutability

If tuple contains list:

```python
t = (1, [2,3])
```

You CAN modify inner list:

```python
t[1].append(4)
```

Tuple reference is fixed, but inner objects may change.

---

### Using tuple where list is required

Beginners sometimes create tuple and later want to append â€” impossible.
