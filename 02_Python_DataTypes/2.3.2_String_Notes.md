
# Python Strings 


## What is a String in Python?

A **string** is a sequence of characters stored as text.

In Python, strings are:

* ordered
* immutable (cannot change individual characters)
* indexed

Examples:

```
"hello"
"Python123"
"@#%"
```

You can use:

```
"double quotes"
'single quotes'
'''multi line'''
```

## String Indexing (0-Based)

Python uses **0-based indexing**.

Range:

```
0  to  len(string) - 1
```

Example:

```

s[0] â†’ 'p'
s[1] â†’ 'y'
s[5] â†’ 'n'
```

Negative indexing also works:

```

s[-2] â†’ 'o'
```

##  String Comparison (Unicode Order)

Python compares strings using **Unicode values**.

Order roughly behaves like:

```
numbers < uppercase letters < lowercase letters
```

### Example

```
"100" < "ABC"    â†’ True
"ABC" < "abc"    â†’ True
"a" > "Z"        â†’ True
```

Why?

Because Unicode value of:

```
'1'  < 'A'  < 'a'
```

Python compares character by character from left to right.

### Difference Between `find()` and `index()`

#### `find()`

```

```

Returns `-1` if not found.

---

#### `index()`

```

```

Raises an **error** if not found.

ðŸ‘‰ Use `find()` when unsure.
ðŸ‘‰ Use `index()` when you expect it to exist.

### Splitting String into List

```

```

Example:

```

```

Result:

```

```

Default delimiter = space.

```
"a b c".split()
```

---

### Joining List into String

```

```

Example:

```

```

Result:

```
"2026-02-14"
```

Important:

ðŸ‘‰ delimiter comes before `.join()`.

## Important Reality About Strings

* Strings are **immutable**.
* Operations create new strings instead of changing original.

Example:

```
s = "hello"
s.upper()
```

Original `s` stays `"hello"` unless reassigned.
