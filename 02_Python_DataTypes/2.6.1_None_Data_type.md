# `None` in Python 

* **`None`** represents **no value** or **nothing** in Python.
* Python does not use `null`; `None` is the equivalent.

```python
x = None
print(type(x))   # <class 'NoneType'>
```

###  Key Points

* Only **one `None` object** exists (singleton).
* It is **immutable**.
* It is **falsy**:

```python
bool(None)   # False
```
