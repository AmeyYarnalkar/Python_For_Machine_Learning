## What is the Purpose of Python Comprehensions?

Normally, beginners write loops like this:

```python
nums = []
for i in range(5):
    nums.append(i)
```

Comprehension lets you write the same thing in **one clean line**:

```python
nums = [i for i in range(5)]
```

ğŸ‘‰ Purpose:

* Write shorter code
* Make transformations easier
* Reduce manual looping boilerplate

## General Syntax of Comprehension

Basic structure:

```python

```


## Simple Example

Normal loop:

```python
squares = []

for i in range(5):
    squares.append(i*i)
```

List comprehension:

```python
squares = [i*i for i in range(5)]
print(squares)   # [0,1,4,9,16]
```

---

## Comprehension With `if` (Filtering)

Goal: keep only even numbers.

```python
nums = [1,2,3,4,5,6]

evens = [x for x in nums if x % 2 == 0]

print(evens)   # [2,4,6]
```

Meaning:

```
Take x
FOR each x in nums
ONLY IF condition is true
```

---

## Comprehension With `if-else`

Structure:

```python
[ value_if_true if condition else value_if_false for item in iterable ]
```

Example:

```python
nums = [1,2,3,4]

labels = ["even" if x%2==0 else "odd" for x in nums]

print(labels)
```

Output:

```
['odd', 'even', 'odd', 'even']
```

Important:

```
if-else comes BEFORE for
```

## List Comprehension

Creates a list.


Most common comprehension type.

---

## Tuple Comprehension (Important Truth)

Python does NOT have a true tuple comprehension.

This:

```python
t = (x for x in range(5))
```

creates a **generator**, not a tuple.

To make tuple:

```python
t = tuple(x for x in range(5))
print(t)
```

---

## Set Comprehension

Uses `{}`.

```python
nums = {x for x in range(5)}

print(nums)   # {0,1,2,3,4}
```

Useful for removing duplicates automatically.

---

## Dict Comprehension

Structure:

```python
{ key_expression : value_expression for item in iterable }
```

Example:

```python
squares = {x: x*x for x in range(5)}

print(squares)
```

Output:

```
{0:0, 1:1, 2:4, 3:9, 4:16}
```
